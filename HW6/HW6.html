Angular is made by Google. huge framework with a lot of built-in features. Has tools like a command line interface for managing projects. Angular uses typescript which is like Javascript but with additional features. Might be a little harder than the others since it requires a more "complex setup"

React is made by Facebook. It is minimalistic and focused on UI-building. Might need to search for libraries with solutions for your issues. Mostly everything is in Javascript and there's less HTML.React seems to be the most popular and most used among the 3.

Vue is an open-source project with a team of collaborators. Focused on code but offers CLI/ more features. For both React and Vue, you decide what is to be rendered on the screen. They all have relatively similar startup performance.
