JavaScript Let vs Var vs Constant

When you declare a variable with var its scope is not limited to the block it was defined, but instead the function, unlike let and const, which are limited to their respective blocks in which they are defined. When you use var to define a global variable it attaches itself to the window object. It is overall better to avoid using the window object.
