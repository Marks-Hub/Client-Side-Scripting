The fat arrow function is part of ecma6, and it can be used write functions in a shorter way, such as: var number0 = function (a) { return a * 2; }; this can be written as: let number1 = a => a * 2; or let number2 = (a) => { return a * 2 };

The array filter method allows you to filter through arrays and create a new one with your desired results. You can filter through an array of numbers and create a new one with only numbers above 3.

The Map method, like the filter method, it has 3 parameters. It can be used to map and array to HTML markups. It can also be used to map an object with a value property. The map method and filter method are chainable so they can be called one after another in a chain.
